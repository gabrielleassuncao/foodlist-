## Descrição do Projeto

**FoodList** é uma aplicação de lista de compras desenvolvida com o objetivo de tornar a organização do dia a dia mais simples, prática e até divertida. O projeto busca transformar uma tarefa comum — montar a lista de compras — em uma experiência intuitiva, acessível e funcional, ajudando o usuário a evitar esquecimentos e a se organizar melhor antes de ir ao mercado.

A ideia do FoodList surgiu a partir de necessidades reais do cotidiano, priorizando usabilidade, clareza e uma navegação amigável. Além de atender aos requisitos acadêmicos da disciplina, o projeto também reflete o interesse dos integrantes em desenvolver soluções digitais úteis, com foco na experiência do usuário e na aplicação prática do desenvolvimento mobile.
